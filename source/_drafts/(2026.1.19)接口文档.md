# 1. 接口类型及约束

**通信协议**: HTTP
**通信方式**: 同步/异步
**通信规范**: RESTful / HTTP页面跳转
**数据格式**: JSON
**基础路径**: `/api/v1` (部分接口除外，如 `/knowledge`)
